使用 Zotero 輔助法學論文撰寫

發表於2024-06-17|更新於2024-06-17|法學研究與寫作工具
|閱讀量:

關於 Zotero 的簡介，以及安裝、收集管理書目、在 Microsoft Word 中如何使用，請參見臺大圖書館參考服務部落格的說明，在此不重複說明。

有了這些先備知識後，剩下的問題就是：有沒有符合臺灣法學引註格式的 citation style 檔案？

對此，有一位律師 GYB，已經將《政大法學評論》的引註語法製作出來，可以參考他的 Github 專案 Mandarin-NCCULR_Chicago。至於如何安裝 .csl 檔案，依據 Zotero 官方文件〈Citation Styles〉「Alternative Installation Methods」一段的說明，只要在 Zotero Style Manager 中，點擊 + 按鈕匯入即可。
